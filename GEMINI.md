# Zombie2 Project Implementation Record

이 문서는 프로젝트의 기능 구현 역사와 설정 사항을 기록하는 공식 문서입니다.

## 1. 프로젝트 설정 및 인프라
- **저장소 관리:** 
  - 로컬 Git 저장소 초기화 및 GitHub 연동 (`origin/main`).
  - Unity 프로젝트 전용 `.gitignore` 파일을 최상위에 생성하여 불필요한 바이너리(Library, Temp 등)를 추적에서 제외.
  - **대용량 파일 관리:** GitHub의 100MB 제한을 초과하는 대용량 클래식 음악 파일(`Assets/Gregor Quendel...`)을 `.gitignore`에 추가하여 백업 안정성 확보.

## 2. 시각 효과 및 URP 최적화
- **셰이더 일괄 수정:** URP(Universal Render Pipeline) 환경에서 에셋들이 보라색으로 깨져 보이던 문제를 해결.
- **주요 수정 대상:**
  - `Explosion` 이펙트 관련 머티리얼 4종.
  - `IceMagicEF` 이펙트 관련 머티리얼 15종.
  - `SnowMan` 눈사람 프리팹 관련 머티리얼 4종 (Body, Head, Eye, Nose).
- **적용 사항:** 기존 레거시 셰이더를 `Universal Render Pipeline/Lit` 및 `Particles/Unlit`으로 전면 교체 및 텍스처 매핑 최적화.

## 3. 핵심 게임 시스템 로직
### 수류탄 (ThrowableGrenade)
- **데미지 면역:** 폭발 시 플레이어 캐릭터는 데미지를 입지 않도록 태그 기반 면역 로직 추가.
- **사운드 최적화:** 폭발 소리가 거리나 3D 공간 제약 없이 선명하게 들리도록 2D 강제 재생 로직 적용.
- **생명주기 관리:** 폭발 이펙트가 무한 반복되지 않도록 생성 2초 후 자동 삭제 처리.
- **밸런스 조정:** 폭발 반경을 3m로 최적화.

### 축음기 유인 시스템 (Gramophone Lure)
- **습득:** 플레이어가 닿으면 인벤토리에 추가되며 점수 획득.
- **설치:** `F` 키 입력 시 즉시 설치 (설치 쿨타임 제거).
- **좀비 AI 연동:** 좀비가 플레이어보다 설치된 축음기(반경 15m)를 최우선 추적하도록 AI 행동 트리 고도화 (10초 유지).

### 눈사람 AI 동료 (SnowMan Companion)
- **소환:** `T` 키로 소환하며 30초의 쿨타임 적용.
- **UI 동기화:** 소환 시 UI 아이콘이 흐려지며 원형 쿨타임(Radial Fill)이 차오르는 시각적 피드백 구현.
- **지능형 이동:** **NavMesh**를 사용하여 장애물을 우회하며 플레이어 오른쪽 0.5m 거리를 정확히 유지.
- **구르기 연출:** 이동 거리에 비례하여 몸통(Mesh)만 물리적으로 회전하고 머리는 똑바로 유지되는 특수 연출.
- **전투 능력:**
  - **슬로우 오라:** 정면 1m 내 좀비를 50% 느리게 만듦.
  - **자폭:** 20초 뒤 강력한 폭발 데미지(500)를 주며 소멸.

## 4. 좀비 시스템 (Zombie Feedback)
- **피격 효과:** 총기 공격을 받을 때마다 맞은 부위와 방향에 맞춰 혈흔 효과(`bloodSprayEffect`)가 터지도록 개선하여 타격감 강화.

## 5. 협업 및 개발 가이드
- **UI 수정 시:** `UIManager`에 연결된 `SnowMan Cooldown Image` 등의 레퍼런스가 누락되지 않았는지 확인할 것.
- **AI 베이킹:** 새로운 장애물 배치 시 반드시 `Navigation` 창에서 다시 `Bake` 해야 눈사람과 좀비가 길을 찾을 수 있음.
